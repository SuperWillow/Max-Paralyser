# AI Detoxifier
An extension that removes google AI overview and blocks AI websites by replacing them with configurable random links (Touhou songs by default).

## Features
+ Disable google AI preview.
+ Redirect AI pages to a random url selected from the possible URLS.

## Configuration
1. Clone the repository

2. Modify banned pages from data/banned_list.json

3. Modify possible random URLS for pages from data/url_list.json

4. Load the extension within your browser

## why does this exist?
This project has two motivations, mainly I made this extension because I was tired of google's AI preview and all the slop that the AI ​​generated for me. I have never used an "agent IDE" but honestly I am not interested and I feel that I waste more time trying to solve problems with AI than I did searching on google for a while or reading documentation, so I decided to choose to simply eliminate the AI ​​from my life. And the second motivation was because I thought it would be VERY FUNNY if someone installed this on a specific pc of one of my engineering colleagues.
